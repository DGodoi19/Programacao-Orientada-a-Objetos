# ☕ Programação Orientada a Objetos (Java)

Este repositório reúne os projetos e exercícios desenvolvidos durante a disciplina de **Programação Orientada a Objetos (POO)** do curso de Ciência da Computação. O foco principal é a aplicação prática de conceitos de engenharia de software utilizando a linguagem Java.

---

## 🏗️ Estrutura do Repositório

O repositório está organizado por módulos de aprendizagem, cada um focando em competências específicas e mosntrando várias aplicabilidades diferentes.

---

## 🧠 Conceitos de Engenharia de Software Aplicados

Para garantir a qualidade e manutenibilidade do código, foram aplicados os pilares da POO:

1. **Encapsulamento:** Uso de modificadores de acesso (`private`) e métodos de acesso (`Getters/Setters`) para proteção de dados.
2. **Abstração:** Modelagem de entidades do mundo real em classes e objetos.
3. **Organização de Projeto:** Separação clara entre arquivos fonte (`src`) e configurações de ambiente.
